3. Reference#

If you’ve completed the tutorial, you are now ready to delve into the internal logic and models used to provide the user-friendly interface. Please note that some sections are still in development and may be subject to changes or removal. This framework is evolving over time, and nothing in the current documentation is considered to be final.

To get started with the internal API, it is recommended to visit the Library API documentation.
